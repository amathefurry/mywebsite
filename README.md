# mywebsite

A silly website that looks like an os thats not yet complete. Made for hackclub. <br>
NO GENERATIVE AI WAS USED IN THE MAKING OF THIS PROJECT, however some snippets were copied and modified from websites like stackoverflow.

<br>
I used the [webOS jam guide from hackclub](https://jams.hackclub.com/batch/webOS) for the frontend, and the [official last.fm api documentation](https://www.last.fm/api) for the backend.
<br>

I also got some help from slack, and i browsed several articles from websites like w3schools, developer.mozilla.org, stack overflow and similar websites, from some of which i also copied some snippets.
<br>

The idea of adding a last.fm window came from listening to music while working on the project. And im really proud i managed to make it work.
The notes window is a short about, while the socials window is a window with links towards some of my social media accounts.
<br>

And although im not proud to admit it, it is my first project i made without generative AI written code, I have been boycotting generative ai due to my concerns regarding its impact on the enviroment. I consider this project to have been a great educational experience for me.
